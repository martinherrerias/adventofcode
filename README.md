## Template

Use `bash new.sh <Y> <N>` to generate a new solution template for day `N` of year `Y`.
After that, `bash new.sh` will create the next day's template in the last used year.

Save your session cookie in a file named `.session` in the repo root to enable fetching of input data. To get the session cookie:

- Log in to `adventofcode.com` in a browser
- Open browser developer tools (F12)
- Go to the Application/Storage tab (Chrome) or Storage tab (Firefox)
- Find the cookie named `session`
- Copy its value (a long hexadecimal string) and save it to a file named `.session` in the repo root
